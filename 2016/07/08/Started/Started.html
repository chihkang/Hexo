Hexo是一款簡易又快速的網誌框架

最近看到許多人都利用它來紀錄和分享一些文章，而且還可以搭配

GitHub來使用，現在就來分享相關心得。

安裝(Install)

首先要安裝

  • Node.js
  • Git

安裝好後，執行下列語法即可完成Hexo的安裝

$ npm install -g hexo-cli

建置(Build)

$ hexo init <folder>

建立好後，稍微檢視一下相關檔案

  • _config.yml 是 Hexo 站台的設定檔 config
    擷取相關資料提供參考
    ```

    Site

    title: Your Blog Title
    subtitle: Your Blog subTitle
    description:
    author: Your Name
    language:
  • zh-tw
  • en
    timezone: Asia/Taipei

URL

If your site is put in a subdirectory, set url as ‘http://yoursite.com/child‘ and root as ‘/child/‘

url: Your Url
root: /
permalink: :year/:month/:day/:title/
permalink_defaults:

> **Note:** Url的部分要特別注意，如果預計使用http://yoursite.com/child/
root也要一併修改成/child/

* [Scaffoids](https://hexo.io/zh-tw/docs/writing.html#鷹架（Scaffold）) 當您建立新文章時，Hexo 會根據 scaffold 來建立檔案。

* source 原始檔案資料夾放置內容的地方。

* [themes](https://hexo.io/zh-tw/docs/themes.html) Hexo 會根據themes來產生靜態檔案。

* [scripts](https://hexo.io/zh-tw/docs/plugins.html#腳本（Scripts）) 在此資料夾內的 JavaScript 檔案會被自動執行。

最後再執行下列語法，就大功告成啦!

```sh
$ cd <folder>
$ npm install
$ hexo server

一旦server執行後使用瀏覽器瀏覽 http://localhost:4000 即可看到部落格運行起來的樣子。
