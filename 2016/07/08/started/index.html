Hexo - Getting Started

2016年7月8日

Hexo是一款簡易又快速的網誌框架

最近看到許多人都利用它來紀錄和分享一些文章，而且還可以搭配

GitHub來使用，現在就來分享相關心得。

安裝(Install)

首先要安裝

  • Node.js
  • Git

安裝好後，執行下列語法即可完成Hexo的安裝

建置(Build)

建立好後，稍微檢視一下相關檔案

  • _config.yml 是 Hexo 站台的設定檔 config
    擷取相關資料提供參考

Note: Url的部分要特別注意，如果預計使用http://yoursite.com/child/
root也要一併修改成/child/

  • package.json 是 Node.js 的套件設定檔。semver 參考

  • Scaffoids 當您建立新文章時，Hexo 會根據 scaffold 來建立檔案。

  • source 原始檔案資料夾放置內容的地方。

  • themes Hexo 會根據themes來產生靜態檔案。

  • scripts 在此資料夾內的 JavaScript 檔案會被自動執行。

最後再執行下列語法，就大功告成啦!

一旦server執行後使用瀏覽器瀏覽 http://localhost:4000 即可看到部落格運行起來的樣子。
